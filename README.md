### About Me

毕业后作为WPF医用上位机软件工程师，后转行Unity游戏客户端。
最近关注的方向：Unity / UE 客户端架构、确定性模拟与网络同步、C# / C++ 工程化，以及用 Python 和 AI Agent 把重复性的开发流程自动化。
